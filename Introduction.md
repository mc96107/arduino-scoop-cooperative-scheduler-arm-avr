# Introduction #

SCoop is a _Simple Cooperative scheduler_ library for the Arduino AVR and ARM platforms providing fifo, task, timer and event handling.


# Details #

> see **SCoop user guide.pdf** file

> class available:

  * **SCoopTask**
  * **SCoopTimer**
  * **SCoopEvent**
  * **SCoopFifo**
  * **SCoopDelay**
  * **SCoopDelayus**

  * **TimerUp**
  * **TimerDown**

  * **Input**
  * **Output**
  * **InputFilter**
  * **InputMem**
  * **OutputMem**

> macro available:

  * defineTaskLoop(,)
  * defineTimerRun(,)
  * defineEventRun(,)
  * defineStack(,)
  * defineFifo(,)
  * yieldPROTECT()
  * yieldATOMIC { }
